# AutoForge 工业智控 AI 平台商业计划书 (2026 版)

## 一、 执行摘要 (Executive Summary)
* **愿景**：成为工业 4.0 时代的自动化软件“超级大脑”与通用底层操作系统。
* **定位**：全球首个基于“多智能体架构 (Agentic AI) + 闭源物理安全沙盒”的工业全栈 AI IDE。
* **核心突破**：将非标自动化设备上位机适配、MES 对接、报警与权限配置的开发周期，从 **3-5 天压缩至 10 分钟**。无论是从零生成，还是无缝接入大客户现有框架，AutoForge 都能提供 100% 安全、零语法错误的 C# 业务源码，彻底释放工业高级研发算力。

## 二、 行业终局痛点 (The Problem)
天下苦“非标”久矣，当前工业自动化面临三大研发深渊：
1. **“手工作坊”式的重复造轮子**：非标设备（如 3C、锂电产线）变型极快。即使有成熟框架，每次新增气缸、修改工艺，都需要工程师手敲“胶水代码”，极易漏写防撞报警与权限，导致代码风格碎裂、现场客诉不断。
2. **MES 对接的无底洞（IT 阻塞 OT）**：面对大厂动辄上百页的异构 MES API 文档，电气工程师沦为“JSON 翻译机”。网络波动处理不当极易导致主线程阻塞，机器现场卡顿甚至停摆。
3. **黑盒信任危机与大模型幻觉**：终端大厂极度反感设备商的“黑盒绑架”，但同时绝对不敢使用通用大模型直接生成的、毫无物理边界控制的概率性代码。

## 三、 颠覆性产品与技术架构 (The Solution)
AutoForge 不是一个简单的代码生成插件，而是一个内嵌 Monaco 编辑器的 **“工业级全栈超级 IDE”**。

* **极致交互（三体联动驾驶舱）**：前端采用 Electron/Web 架构。左侧 SFC 物理拓扑画布，中控 C# 代码手术台，右侧 AI Team Lead 实时对齐。图、码、AI 原子级共振。
* **首创“白盒交付 + 黑盒锁死”双层安全架构**：
    * **白盒应用层（极速自愈生成）**：LangGraph 多智能体图网络（总工/运动/视觉/IT 专家）后台并行。图纸、表格输入后，自动生成 100% 开源、符合客户内部规范的 C# 源码。
    * **黑盒安全底座（物理防御 DLL）**：无论 AI 怎么写，底层 C++ 沙盒执行微秒级干涉运算与 IT/OT 异步网络隔离。**绝对不撞机，绝对不卡机！**
* **非侵入式增效引擎（针对大客户）**：不推翻客户现有框架，自动读取其组件库规范，仅负责生成枯燥的胶水代码、状态机逻辑与自带断点续传的 MES 异步通信模块。

## 四、 核心护城河 (Competitive Moat)
面对巨头环伺，我们构筑了三层结构性防御：
1. **物理责任泥潭 (vs. IT 巨头如 Copilot/Cursor)**：IT 大厂不敢为物理撞机承担法务责任，永远不敢涉足底层执行。我们的 C++ 物理防撞沙盒是我们敢于接管工厂手脚的底气。
2. **中立的生态霸权 (vs. OT 巨头如 西门子/汇川)**：传统硬件巨头做软件是为了卖自家封闭的硬件。AutoForge 是绝对中立的“工业安卓”，通过 JSON Skill 插件兼容万国牌硬件，打破 Vendor Lock-in。
3. **不可逆的算力习惯 (vs. 抄袭者)**：核心智能体路由与纠错闭环放于云端。一旦工程师习惯了毫无摩擦的 AI 极速开发，所有工程图纸与变量资产沉淀在云端，替换成本极高。

## 五、 商业模式 (Business Model) —— 极具爆发力的 PLG 算力税
彻底放弃阻碍裂变的“单机硬件税”，转向高频、高毛利的 SaaS + API 算力模式，重塑估值模型：

* **社区探索版 (永久免费 - 影子 IT 渗透)**
    * **定位**：学生、底层电工接私活、打样验证。
    * **策略**：赠送基础晶体算力。底层 DLL 带有 **“2 小时运行限制”水印**，机器完美跑两小时后强制要求升级。用极致体验培养断点式依赖。
* **工业专业版 Pro (3,899 元/人/年 - 绝对利润引擎)**
    * **定位**：中小型系统集成商、大厂研发部。在主管签字权限内，极速成单。
    * **权益**：解锁纯净版无限分发 DLL（不收单机费），**内含 30,000 算力晶体**，消除算力焦虑。
* **高毛利算力消耗 (Pay-as-you-go)**
    * 小模块微调扣 5 晶体，整机架构生成扣 100 晶体。复杂非标项目带来的高频重构，将催生源源不断的超额晶体充值（算力毛利率 >80%）。
* **集团私有版 (百万级买断)**：针对富士康、比亚迪等大厂，提供本地私有大模型部署与源码级白盒审计。

## 六、 市场规模与 GTM 策略 (Market & GTM)
* **珠三角基本盘 (TAM / SAM)**：仅深莞广地区就有超 2 万家集成商，年新增非标设备 100 万台。这是全球最卷、对降本增效最渴望的“海眼”。
* **GTM 特洛伊木马战术**：
    1. **工程师起义**：销售无需死磕老板，让免费的社区版直接潜入工程师的电脑，解决他们最头疼的 MES 联调与胶水代码。
    2. **降维演示打单**：带着 AutoForge 进厂，喝杯茶的时间跑通客户历史最复杂的 MES API 对接，当场逼死传统外包，极速转化 3899 元 Pro 坐席。

## 七、 战略演进路线 (Roadmap)
* **Phase 1 (现在-半年)：边缘突围与生态定盘**。主打 IT/OT 数据网关、标准机控制及成熟框架的非侵入式代码增强，利用“晶体算力”模式极速铺开市占率。
* **Phase 2 (1年)：高阶视动统治**。开放高阶视觉补偏与多轴联动宏指令，统治 3C 与锂电复杂柔性产线。反向逼迫硬件大厂主动提交 Skill 插件。
